  • ✓

  • ✓
    Syncs with your coworking platform. Pull live member and booking data from Nexudus, Cobot, or OfficeR&D.

    Morning digest in your inbox. A short summary of everything processed overnight, so you start the day knowing what's been handled.

    Set up from a spreadsheet menu. No config files, no code. Your API keys and email addresses go in through a guided prompt in Google Sheets.
  • ✓
    Free to run. It lives inside Google's free tier. No subscription, no server, no bill at the end of the month.

Takes about 30 minutes to set up

No coding. Everything happens from inside a Google Sheet.

  1. 1

    Copy the project from GitHub

    Clone the repo and deploy it to Google Apps Script using clasp. The README walks you through it step by step.

  2. 2

    Enter your credentials from the menu

    Open the sheet, click Email Triage > Configure API Keys. It walks you through your Gemini key, inbox address, and platform credentials one at a time.

  3. 3

    Run the setup check

    Click Test Setup to confirm Gmail, Sheets, and the AI are all connected properly.

  4. 4

    Send yourself some test emails

    The menu includes a safe test mode that sends eight realistic member emails so you can see it work before going live.

What you need

  • A Google account
  • A free Gemini API key from Google AI Studio
  • A Gmail inbox for member emails
  • About 30 minutes
The Gemini free tier covers around 1 million tokens a month. That's a lot of emails. You won't hit it.
