Quick Start

Up and Running in 30 Minutes

No coding required — everything is configured from inside the Google Sheet

  1. 1

    Copy the Script to Apps Script

    Clone or copy the files from GitHub into a new Google Apps Script project linked to a Google Sheet.

  2. 2

    Configure via the Sheet Menu

    Open the sheet, click Email Triage > Configure API Keys. A step-by-step wizard collects your Gemini key, inbox address, and platform credentials — no file editing needed.

  3. 3

    Run Setup & Go Live

    Click Test Setup to verify everything works, then Schedule Auto-Processing to start the 15-minute cycle.

  4. 4

    Send Test Emails

    Use Send 8 Test Emails from the menu to see categorisation, labels, and draft replies in action before going live.

What You Need

  • Google Account (free)
  • Gemini API Key (free from Google AI Studio)
  • Gmail inbox for member emails
  • 30 minutes setup time
  • Zero coding knowledge
The system runs entirely within Google's free tier. No servers, no hosting, no monthly fees. The Gemini free tier includes 1 million tokens per month — more than enough for a busy coworking inbox.
